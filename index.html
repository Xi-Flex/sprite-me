jquery.spriteMe

  • maxframe + loop + reverse : la loop revient sur le max frame jusque la frame 0. C est pour pouvoir jouer à l'envers une animation quand on a maxframe défini.

Utilisation

Définir une div avec width et height plus un background-image. la div affichera donc la 1er frame du sprite (cela peut etre modifié lors de l'initialisation du plugin).
Le fichier image peut avoir une ou plusieurs colonnes, peut importe. le sprite sera affiché de gauche à droite, ligne par linge. 

#lookbook{
  width: 807px;
  height: 400px;
  background-image: url(../images/sprite-lookbook.png);
}

initialisation basic et avec parametres

$("#lookbook").spriteMe();

// avec params
$("#lookbook").spriteMe({
	fps:30,
	loop:true,
	autoplay:false,
	reverse:true,
	maxframe: 52,
	gotoframe:32,
	complete: function(){  
		console.log('This is the end.');
	}
});

Parametre par default

•fps: 12, // nombre d'image par seconde
•loop: false, // tourne en boucle : BOOLEAN
•autoplay: true, // BOOLEAN
•gotoframe:false, // BOOLEAN OU INT
•reverse:false, // joue le sprite à l'envers :BOOLEAN
•maxframe: false, // si on a un sprite qui a 10 frames mais que notre animation en a que 9. BOOLEAN ou INT
•complete: function () {}, // execute un bloc de code quand l'animation arrive à la fin ET QUE LOOP = FALSE.

Méthode / API

Les méthodes peuvent étre appellées pendant la lecture de l'animation, ou non.

	// les deux methodes sont identiques, play est un racourci pour resume
	$("#lookbook").spriteMe("play");
	$("#lookbook").spriteMe("resume");
	
	// stop l'animation
	$("#lookbook").spriteMe("stop");
	
	//relance l'animation
	$("#lookbook").spriteMe("restart");
	
	//change les fps : INT
	$("#lookbook").spriteMe("fps",30);
	
	//saute vers la frame : INT
	$("#lookbook").spriteMe("gotoframe",30);

	// inverse le sens de lecture du sprite
	$("#lookbook").spriteMe("reverse");

	// change le nombre de frame de l'animation INT
	$("#lookbook").spriteMe("setmaxframe",42);

	// supprime la limitation de frame: BOOLEAN
	$("#lookbook").spriteMe("setmaxframe",false);
	
	// change l'etat de la répétition automatique
	$("#lookbook").spriteMe("setloop",false);
